About

This is a simple web app that can be used to check the integrity of a file. You can upload a file and generate a hash for it. You can also check the integrity of a file by comparing the hash of the file with the hash you have generated.

  • It is open source project.
  • It does not include any server side processing so no data is shared across network.
  • You can also upload generated hash in public domain to store the hashed file as orignal data can not be regenerated.

This app is created for educational purposes. You can find the source code of this app on GitHub.
